# Landing Page - Marinho Mendes Advogados

## 📋 Descrição do Projeto

Landing page profissional e institucional para o escritório Marinho Mendes Advogados, especializada em Aposentadoria por Tempo de Contribuição 2025. Desenvolvida com foco em alta conversão e total conformidade com o Código de Ética da OAB.

## 🎯 Características Principais

### ✅ Conformidade Ética
- Linguagem sóbria, informativa e institucional
- Conformidade total com o Provimento 205/2021 da OAB
- Evita termos promocionais e promessas de resultado
- Comunicação compatível com a dignidade da advocacia

### 🎨 Design e UX
- Design responsivo (desktop e mobile)
- Formulário otimizado para alta conversão
- Paleta de cores profissional (azul escuro, cinza, branco)
- Tipografia Roboto
- Animações sutis e profissionais

### 📱 Funcionalidades
- Header fixo com navegação suave
- Formulário de contato com validação
- Link discreto do WhatsApp fixo
- Links reais das redes sociais
- Otimizações de performance e SEO

## 🚀 Como Usar

### 1. Hospedagem
1. Extraia todos os arquivos do zip
2. Faça upload de todos os arquivos para seu servidor web
3. Certifique-se de que o arquivo `landing_page.html` está na raiz
4. Acesse via navegador

### 2. Configurações Necessárias

#### Formulário de Contato
O formulário está configurado mas precisa de um backend para processar os dados. Opções:
- Integrar com serviços como Formspree, Netlify Forms ou EmailJS
- Desenvolver backend próprio em PHP, Node.js ou Python
- Usar ferramentas como Zapier para automação

#### SSL/HTTPS
- Configure certificado SSL para segurança
- Essencial para formulários e credibilidade

### 3. Personalização

#### Cores
As cores principais estão definidas no CSS:
- Azul escuro: `#1C2E4A`
- Cinza claro: `#E5E5E5`
- Branco: `#FFFFFF`

#### Conteúdo
- Textos podem ser editados diretamente no HTML
- Imagens podem ser substituídas mantendo os mesmos nomes
- Links das redes sociais já estão configurados com os perfis reais
